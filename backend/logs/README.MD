## Logs

This folder holds all the exceptions that occur during runtime.

The errors are displayed in this format:

```
**********************************
*        ERROR LOG ENTRY        *
**********************************
Timestamp: {{TIMESTAMP}}
Error: {{ERROR_MESSAGE}}

Request Info:
Method: {{REQUEST_METHOD}}
URL: {{REQUEST_URL}}
Headers: {{REQUEST_HEADERS}}
Body: {{REQUEST_BODY}}

Stack Trace:
{{STACK_TRACE}}

**********************************
```
